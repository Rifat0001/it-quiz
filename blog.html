Question-1: What is the difference between map(), filter(), foreach() and find()?

Answer:

  • The map() method loops every element of an array and match with required condition and finally return a new array.
  • filter() method check any condition from an array and return a new array after checking all matching conditions.
  • forEach() method is used to execute the same code on every element in an array but it does not change the array or array like objects and it returns undefined.
  • find() method check any condition from an array like filter but it returns the first element which match with the condition.

Question-2: What is the difference between local storage, session storage and cookies?

Answer:

  • Local storage store the most recent data and it is not deleted when the browser is closed.
  • Session storage store data but only for current session, it is deleted after closing the browser.
  • Cookies store data on client side in a text file.

Question-3: What is Event Loop and its function?

Answer:

  • The Event Loop is a programming design pattern which waits for and dispatches events in a program. It is just a guardian who keeps a good communication with Call Stack and Callback Queue. The functions of Event Loop is given below:
  • 1. Code that waits for and dispatches events
  • 2. Monitors the call stack and callback queue
  • 3. Handles the events for its environment
  • 4. Breaks down asynchronous code into ticks
